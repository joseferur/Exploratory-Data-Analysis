Netflix Content Analysis

- Bussiness Problem
    Content strategy is key in the competition between streaming platforms. The understanding of content distribution by type, genre, country, and ratings, can become a major aid to:
        Investment strategy
        Regional expansion
        Content creation decisions
        Audience targeting

- Analysis Goal
    Exploratory data analysis on Netflix's catalog intents to identify:
        Content rating distribution
        Most common genres
        Distribution of Movies vs TV shows
        Top contributing countries
        The growth of trends over time

- Dataset
    Netflix Titles dataset from Kaggle

- Tools Used
    Python
    Pandas
    Numpy
    Matplotlib
